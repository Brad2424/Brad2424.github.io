My Work

Employee Directory

(mobile-friendly)

In this project I used a Random User Generator API to grab information for 12 random “employees,” and then used that data to build a prototype for a company employee directory.

I also created a modal window with more detailed information when an employee is clicked, using vanilla JS.

Github page

Mock Dashboard

(mobile-friendly)

In this project I built the client-side of a web dashboard homepage, complete with JavaScript-driven charts and graphs.

I was given a design mockup to follow, as well as svg icons in which to customize to suit the dashboard.

Github page

Image Gallery

(mobile-friendly)

In this project, I created an interactive photo gallery using vanilla JavaScript and jQuery. Search bar is included where photos will hide and show depending on user input.

Also, when a user clicks on a thumbnail, the photo will display in a lightbox with back and previous arrows to cycle through photos.

Github page

Custom Stylesheet

(mobile-friendly)

In this project I created rules to style a pre-existing webpage using only element class names already established in the HTML. I learned to utilize SASS for this project, using partials, variables, extends, and mixins to apply the styles and classes to the style guide page.

I now have a SASS micro-framework to quickly prototype other websites.

Github page

Form Example

(mobile-friendly)

In this project, I built a responsive, mobile-friendly registration form using a wide variety of HTML form input types and attributes.
